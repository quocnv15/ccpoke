配置

ccpoke 将配置存储在 ~/.ccpoke/config.json。你可以直接编辑或重新运行安装向导。

文件: ~/.ccpoke/config.json

键 类型 默认值 说明
telegram_bot_tokenstringrequiredBotFather Token
user_idintegerrequired你的 Telegram 用户 ID
hook_portinteger9377本地 hook 服务器端口
hook_secretstringautoHMAC 验证密钥
localestring"en"语言: en / vi / zh
agentsstring[]["claude-code"]活跃代理
projectsarray[]{name, path} 条目

示例

{
  "telegram_bot_token": "123456:ABC-xxx",
  "user_id": 123456789,
  "hook_port": 9377,
  "hook_secret": "auto-generated-hex",
  "locale": "zh",
  "agents": ["claude-code", "cursor"],
  "projects": [
    { "name": "my-app", "path": "/home/user/code/my-app" }
  ]
}

提示：随时运行 ccpoke setup 重新配置。
