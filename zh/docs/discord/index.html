Discord 使用

使用 Discord 通道设置 ccpoke 后你能做的一切。

斜杠命令

两个斜杠命令：

命令 说明
/sessions 查看所有正在运行的 AI 代理会话。选择一个进行聊天或关闭。
/projects 查看项目列表。选择项目和代理启动新会话。

通知

当 AI 代理完成任务时（stop hook 触发）

你会收到 DM 嵌入通知：项目名称、代理类型、任务摘要、模型名称。按钮："查看详情" 打开完整响应，"聊天" 打开模态框发送后续提示。

双向聊天

点击通知上的 "聊天" 或 /sessions → 选择 → Chat

点击 Chat 按钮打开模态框。输入提示 — ccpoke 通过 tmux send-keys 注入代理。回复 elicitation 消息以回答开放式问题。

→ 点击 💬 聊天

[模态框: 发送提示到 my-app]
> 总结最近的变更

✓ 已发送到会话

需要 tmux（macOS / Linux）。Windows 暂不支持双向聊天。

权限处理

当代理需要权限时（写入文件、执行 bash 命令等）

代理暂停并发送 DM，包含工具名称和摘要。点击 Allow 或 Deny。ccpoke 注入响应。10 分钟后过期。

🔐 权限请求

工具: bash
命令: rm -rf node_modules && npm install

[✅ 允许]  [❌ 拒绝]

单选/多选问题以按钮行显示。开放式问题需要你回复消息输入答案。
