功能

不仅仅是通知

推送通知

AI 代理完成 → 立即推送通知。无轮询，无延迟。

双向交互

从 Telegram 与 AI 代理聊天。查看活跃会话、发送提示、回答问题 — 无需触碰电脑。

多会话

同时管理多个 AI 代理会话 — 切换会话、并行监控、每会话独立上下文。

安装

2 步，30 秒

需要 Node.js ≥ 20。

1

创建 Bot Token

Telegram: 打开 @BotFather → /newbot → 复制 Token。

Discord: Developer Portal → New Application → Bot → 复制 Token。

Slack: Slack API → Create New App → From scratch → 命名应用并选择工作区 → OAuth & Permissions → 添加 chat:write 权限 → 滚动到 OAuth Tokens → 生成并复制 Bot User OAuth Token。

2

运行一个命令
